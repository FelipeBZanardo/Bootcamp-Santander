Operadores:
	São sinais utilizados para manipular um valor ou validar uma condição

tipos:
	- atribuição
	- aritmética
	- comparação
	- lógica
	- condicional

1-) Atribuição:
	 = 	x = y		x recebe y
	+=	x += y 		x = x + y
	*=	x *= y		x = x * y
	/=	x /= y		x = x / y
	%= 	x %= y		x = x % y

2-) Aritmética:
	+			adição
	-			subtração
	*			multiplicação
	**			exponenciação
	/			divisão
	%			módulo - resto da divisão
	++			incrementar(soma 1)
	-- 			decrementar(subtrai 1)

3-) Comparação:
	==			igual a						2 == '2'	True
	===			mesmo valor e mesmo tipo			2 === '2'	False
	!=			diferente
	!==			valor e tipos diferentes
	>			maior que
	<			menor que
	>=			maior ou igual
	<=			menor ou igual

4-) Lógica:
	&&			and
	||			or
	!			not

5-) Condicional:

	(condition) ? expression1 : expression2				Se a condição for verdadeira, executa a expressão1, caso contrário, expressão2

	if (condition){
		expression1}
	else{
		expression2}
